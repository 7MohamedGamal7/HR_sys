# نظام التصميم الموحد

## الأهداف
- توحيد تجربة المستخدم والشكل البصري عبر جميع الشاشات والنماذج.
- تحسين سهولة الاستخدام والوصول والتوافق مع جميع الأجهزة.
- تسهيل الصيانة والتطوير المستقبلي عبر قواعد ثابتة وواضحة.

## الأساسيات البصرية
- الخط الأساسي: Cairo بعرض مختلف للأوزان (300/400/600/700).
- اتجاه الصفحة: `dir="rtl"` ودعم Bootstrap RTL.
- ألوان رئيسية:
  - primary: `#2c3e50`
  - secondary: `#3498db`
  - success: `#27ae60`
  - danger: `#e74c3c`
  - warning: `#f39c12`
  - info: `#16a085`
  - light-bg: `#ecf0f1`

## القواعد العامة للنماذج
- استخدام Bootstrap 5 مع Crispy Forms (`bootstrap5`).
- الحقول:
  - `form-label` وزن 600، لون نص داكن.
  - `form-control` نصف قطر 8px، ارتفاع أدنى 42px.
  - تباعد عمودي ثابت عبر `.form-group` و`.mb-3` بقيمة 1rem.
- التركيز والوصول:
  - `:focus-visible` يضيف إطارًا واضحًا لعناصر الإدخال والأزرار.
  - الحفاظ على نسب تباين جيدة بين النص والخلفية.
- الإجراءات (الأزرار):
  - استخدام شريط إجراءات موحّد عبر `.form-actions` بمحاذاة البداية وفواصل 10px.
  - وضع الأزرار أسفل النماذج داخل البطاقات مع حاجز علوي خفيف.

## تخطيط الصفحات
- استخدام `card` للعناوين والمحتوى النموذجي، مع تباعد داخلي مناسب.
- استخدام `container-fluid` و`row` و`col-md-*` لتخطيط متجاوب.
- توحيد الرسائل والتنبيهات عبر `alert` مع `btn-close` ودعم الإخفاء.

## القوالب الأساسية
- القالب العام `templates/base.html` يتضمن:
  - Bootstrap RTL، Font Awesome، خطوط Cairo.
  - شريط جانبي للتنقل ثابت مع قوائم فرعية.
  - رأس علوي Sticky، منطقة محتوى، وتذييل.
  - أنماط موحّدة للأزرار والجداول والتنبيهات والنماذج.
- القالب `hr_app/templates/hr_app/base.html` تمت مواءمته بنفس الموارد (Bootstrap RTL، Cairo) وبنفس أنماط النماذج.

## إرشادات التطوير
- عند إنشاء نموذج جديد:
  - استخدم Crispy Forms مع `FormHelper` و`Layout` و`Row/Column`.
  - أضف `FormActions` في الأسفل للأزرار الرئيسة.
  - تأكد من أن الحقول غير المطلوبة لا تحتوي `required` في الواجهة.
- عند إنشاء صفحة قائمة/جدول:
  - استخدم `table` مع رأس بخلفية فاتحة وخط غامق.
  - أضف بحث/تصفية أعلى الجدول داخل `card-header`.

## الوصول والنفاذ الرقمي
- دعم لوحة المفاتيح عبر `:focus-visible` لعناصر التحكم.
- استخدام تسميات واضحة للعناصر وفواصل جمل مناسبة.
- تجنب الاعتماد على اللون فقط للتعبير عن الحالة.

## الأداء والاستجابة
- استخدام Bootstrap CDN وخطوط Google لتحسين التحميل.
- تقليل الأنماط المخصصة والاكتفاء بالتخصيصات العامة.

## الاختبار
- راجع الصفحات على أجهزة مكتبية ومحمولة وبمتصفحات حديثة.
- استخدم أدوات المتصفح لفحص التباين والتركيز.

## الصيانة
- أضف أي تخصيصات إضافية على مستوى `templates/base.html` أو `hr_app/static/css/style.css` للمحافظة على الاتساق.
- تجنب الأنماط المحلية داخل الصفحات إلا للضرورة.
